# Demo Telegram Mini App booking calendar

Красивый Mini App календарь для демо-бота базы отдыха.

## Что умеет

- открывает календарь как Telegram Mini App;
- переключает месяцы кнопками;
- отмечает занятые даты;
- выбирает дату заезда и дату выезда;
- не дает выбрать прошлые даты;
- не дает выбрать период, где есть занятые дни;
- отправляет выбранные даты обратно в чат-бота;
- показывает итоговый период и количество ночей;
- после выбора дат предлагает домики;
- показывает фотографии домиков и услуг, если они добавлены в папку `photos`;
- считает стоимость проживания;
- показывает демо-оплату;
- отвечает на частые вопросы: адрес, цены, территория, условия, контакты.

## Запуск

```powershell
pip install -r requirements.txt
$env:BOT_TOKEN="TOKEN_FROM_BOTFATHER"
$env:WEBAPP_URL="https://your-domain.com"
python demo_bot.py
```

Если нужно отправлять демо-бронь в админский чат, добавьте `ADMIN_CHAT_ID`:

```powershell
$env:ADMIN_CHAT_ID="123456789"
```

## Сценарий демо

1. Пользователь нажимает `/start`.
2. Бот предлагает выбрать страну: Казахстан или Россия.
3. Бот показывает кнопку `Выбрать даты` и FAQ-кнопки.
4. Пользователь открывает Mini App календарь.
5. Выбирает заезд, выезд и домик.
6. Mini App считает итоговую стоимость в валюте выбранной страны.
7. Бот получает готовый выбор и показывает одну сводку.
8. Пользователь нажимает `Демо-оплата`.
9. Бот показывает следующий демо-шаг и информацию для гостя.

## Синее меню Telegram

При запуске бот настраивает команды в синем меню Telegram:

```text
/start - Запустить демо
/book - Открыть бронирование
/support - Связаться с поддержкой
```

## Фото для полноценной демо-версии

Создайте или положите реальные фотографии в папку `photos`:

```text
photos/house_2.png
photos/house_4.png
photos/house_6.png
photos/gazebo.png
photos/bathhouse.png
photos/pool.png
photos/bbq.png
```

После этого бот автоматически начнёт отправлять карточки с фотографиями. Если какого-то файла нет, бот покажет обычную текстовую карточку.

Для красивой демонстрации лучше использовать горизонтальные фотографии примерно 1200x800 или 1600x1000, без тяжёлого веса файла. Оптимально сжать каждое изображение до 300-800 КБ.

## Сжатие фотографий

Если PNG-фотографии грузятся долго, положите исходники в:

```text
webapp/photos/
```

Потом выполните:

```powershell
python -m pip install Pillow
python optimize_photos.py
```

Скрипт создаст лёгкие JPG-файлы здесь:

```text
webapp/photos_optimized/
```

Именно папку `photos_optimized` нужно загрузить на хостинг вместе с `index.html`, `styles.css` и `script.js`.

## Mini App календарь

В папке `webapp` лежит более красивый календарь для Telegram Mini App:

```text
webapp/index.html
webapp/styles.css
webapp/script.js
```

Чтобы открыть Mini App из Telegram, файлы нужно разместить на HTTPS-хостинге, например Vercel, Netlify, Cloudflare Pages или временно через ngrok. После этого перед запуском бота укажите ссылку:

```powershell
$env:WEBAPP_URL="https://your-domain.com"
python demo_bot.py
```

Если `WEBAPP_URL` не указан, бот не будет показывать обычный календарь. Он напишет, что нужно подключить HTTPS-ссылку на Mini App.

## Где подключить Google Sheets

Сейчас занятые даты заданы в `demo_bot.py`:

```python
BUSY_RANGES = [
    (date(2026, 7, 5), date(2026, 7, 8)),
    (date(2026, 7, 14), date(2026, 7, 16)),
    (date(2026, 8, 1), date(2026, 8, 4)),
]
```

В реальном боте эти диапазоны нужно получать из листа `Bookings` в Google Sheets.
